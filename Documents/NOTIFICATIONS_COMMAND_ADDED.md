## ✅ New Command: `/notifications`

**Purpose:** Show recent signal notifications history

**Usage:**
```

```

📋 /signals - Lihat semua sinyal

---

## 📝 Command Summary

| Command | Function |
|---------|----------|
| `/signals` | Show current trading signals for ALL watched pairs |
| `/notifications` | Show recent signal notifications history (last 10) |
| `/signal <pair>` | Detailed signal for ONE pair |
| `/watch <pair>` | Add pair to watchlist |
